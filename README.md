# selfbot-js
discord.js-selfbot-v13を使ったTOS違反SELFBOT

# 何これ

### このコードをもっと良くできる方はpull reauestなりissuseお願いします

いろんな機能を備えたセルフボット TOS違反なので絶対に使わないアカウントで、秘密情報を保存してないアカウント(もしかしたらあなたのアカウントが死ぬかもしれません)、で使用しましょう

タイムアウトされた時の例外処理できてないのでそれが弱点です

いろんな機能があります

コード読んで理解してください()
